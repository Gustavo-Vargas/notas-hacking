
## Description
Using netcat (nc) is going to be pretty important. Can you connect to fickle-tempest.picoctf.net at port 59160 to get the flag?
## Solución 


- Usar al terminal de Webshell dentro de picoCTF 
- Realizar comando de ` nc fickle-tempest.picoctf.net 51079 > salida.txt ` para crear lo que esta dentro en una archivo llamado salida.txt
- abrir el archivo creado con el comando de ` grep "picoCTF" salida.txt `


```
	picoCTF{digital_plumb3r_1eBfC512}
```

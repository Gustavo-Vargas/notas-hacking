
## Description
What is 0x3D (base 16) in decimal (base 10)?

## Solución 
Convertir 0x3D en hexadecimal a su representación en decimal
- Usar una pagina web hex to ascii
- Usar una herramienta web como cyberhef

```
  picoCTF{61}
```

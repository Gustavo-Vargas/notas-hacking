
## Description
There is a nice program that you can talk to by using this command in a shell:$ nc wily-courier.picoctf.net 53584, but it doesn't speak English...
## Solución 

- Usar la terminal de WebShell que esta en picoCTF
- Con el comando de ` nc wily-courier.picoctf.net 53584 ` para que nos de el listado de números 
- Hacer la transformación de numero a ascii con  la herramienta  cyberhef


```
  picoCTF{g00d_k1tty!_n1c3_k1tty!_d5d88}
```
